# Introduction

::: warning Fork notice
olpat is a fork of [Taplo](https://github.com/tamasfe/taplo). This project is **not under active development**.
:::

olpat CLI aims to be an one stop shop tool for working with TOML files via the command line. The features include validation, formatting, and querying TOML documents with a [jq](https://stedolan.github.io/jq/)-like fashion.
