# NPM

The [default](./binary.md#default-build) build of olpat is published to NPM as [`@olpat/cli`](https://www.npmjs.com/package/@olpat/cli). You will need [Node.js](https://nodejs.org/en/) (tested with 16 and up) in order to use it.

## Install with npm

```sh
npm install -g @olpat/cli
```

## Install with Yarn

```sh
yarn global add @olpat/cli
```

## Install with pnpm

```sh
pnpm install -g @olpat/cli
```

## Run without installing

Alternatively you can run it once "without installing":

```sh
npx @olpat/cli --help
```

```sh
pnpm dlx @olpat/cli --help
```
